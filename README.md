# Dotfiles
The bestest of the dotfiles.

## Install

1. Clone 
    - SSH:  
    ```git clone --recurse-submodules git@github.com:Sigurs/dotfiles.git ~/.dotfiles```
    - HTTP:   
    ```git clone --recurse-submodules https://github.com/Sigurs/dotfiles.git ~/.dotfiles```
2. Run ```cd ~/.dotfiles && ./install.sh```
